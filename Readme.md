# OpenBilim - Образовательная платформа для проведения тестов

Платформа с открытым исходным кодом для создания и прохождения образовательных тестов.

## Запуск проекта

```bash
gradle run --args='<Желаемый порт для запуска сервера>'
```

## API Endpoints

### Аутентификация пользователя

**POST** `/auth`

**Тело запроса:**
```json
{
    "login": "Электронная почта",
    "password": "пароль"
}
```

**Ответ:**
```json
{
    "fio": "Фамилия Имя Отчество",
    "user_token": "Токен Пользователя",
    "group": "Группа"
}
```

### Создание тестовой сессии

**POST** `/startTest`

**Тело запроса:**
```json
{
    "user_token": "Токен пользователя",
    "test_id": "Идентификатор теста"
}
```

**Ответ:**
```json
{
    "subject": "Название дисциплины",
    "testName": "Название теста",
    "tasksNumber": "Количество заданий",
    "sessionToken": "Токен сессии"
}
```

### Прохождение тестирования

**GET** `/{session_token}/getTask`

**Ответ:**
```json
{
    "taskText": "Текст задания",
    "options": ["Массив", "Вариантов", "Ответа"]
}
```

*Примечание: Сериализация типа задания будет реализована в следующих версиях*

### Ответы на задания

#### Текстовое задание

**POST** `/textAns/ans`

**Тело запроса:**
```json
{
    "userToken": "Токен пользователя",
    "answer": "Текст ответа"
}
```

#### Задание с выбором одного варианта

**POST** `/singleChoise/ans`

**Тело запроса:**
```json
{
    "userToken": "Токен пользователя",
    "answer": "Вариант ответа в виде содержания выбранного варианта"
}
```

#### Задание с выбором нескольких вариантов

**POST** `/multiChoise/ans`

**Тело запроса:**
```json
{
    "userToken": "Токен пользователя",
    "answer": "Варианты ответа в виде содержания выбранных вариантов"
}
```
