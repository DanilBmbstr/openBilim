OpenBilim - Образовательная платформа для проведения тестов с открытым исходным кодом



На данный момент раззработан основной функционал api. 

Запуск производтся через gradle run --args='<Желаемый порт для запуска сервера>'

Со списком вызовов API вы можете ознакомиться далее:


Начало работы:

POST Аутентификация пользователя:
/auth

{
    "login": "Электронная почта",
    "password": "пароль"
}

Возвращает: {"fio":"Фамилия Имя Отчевство","user_token":"Токен Пользователя","group":"Группа"}


POST Создание тестовой сессии:
/startTest
{
    "user_token": "Токен пользователя",
    "test_id": "Идентификатор теста"
}
Врзвращает: {"subject":"<Название дисциплины>","testName":"<Название теста>","tasksNumber":<Количевство заданий>,"sessionToken":"<Токен сессии>"}


GET Прохождение тестирования:
<токен сессии>/getTask

Возвращает: {"taskText":"<Текст задания>","options":["Массив","Вариантов","Ответа"]}
Сериализация типа задания будет реализована в следующях версиях


POST ответ на текстовое задание 
textAns/ans
{
"userToken" : "Токен пользователя",
"answer" :"Текст ответа"

}

POST ответ на задание с выбором варианта
singleChoise/ans
{
"userToken" : "Токен пользователя",
"answer" :"Вариант ответа в виде содержания выбранного варианта"

}


POST ответ на задание с выбором нескольких вариантов
multiChoise/ans
{
"userToken" : "Токен пользователя",
"answer" :"Варианты ответа в виде содержания выбранных вариантов"

}
